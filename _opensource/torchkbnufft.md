---
title: "Torch KB-NUFFT"
excerpt: "A high-level package for computing Non-Uniform Fast Fourier Transforms with PyTorch."
---

I developed [torchkbnufft](https://github.com/mmuckley/torchkbnufft), a high-level PyTorch implementation of the non-uniform Fast Fourier Transform using Kaiser-Bessel interpolation. Since the package is implemented entirely at the high level, no compilations are necessary to use it and it is supported on any hardware that PyTorch supports.
